# Contributing

Thanks for checking out the project — feedback and contributions are welcome.

## Ways to Contribute

You can help by:

* Reporting bugs
* Suggesting improvements or new features
* Sharing how you're using the tool
* Submitting pull requests

## What Happens When You Contribute

If you share feedback, ideas, or code:

* I’m free to use it, modify it, and include it in the project
* It may be used in future versions or commercial products
* You won’t be credited as an owner of the project

This keeps the project simple to manage and avoids ownership confusion later.

## Code Contributions

If you submit code (like a pull request):

* Make sure it’s your own work
* Keep it consistent with the style of the project
* Keep changes focused and easy to review

## Keep It Practical

This isn’t a heavily managed open-source project (yet).
Small, clear contributions and real-world feedback are the most helpful.

## Be Respectful

Keep discussions constructive and focused on improving the tool.
